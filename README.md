## 📂 주요 폴더 및 파일 구성

이 프로젝트를 협업하거나 빌드하기 위해 필수적인 파일들의 목록과 중요도입니다.

| 폴더/파일 | 설명 | 중요도 |
| :--- | :--- | :--- |
| **`User/`** | **핵심 소스코드.** (`main.c`, `CTRL.c` 등) 프로젝트의 가장 중요한 부분입니다. | ⭐⭐⭐⭐⭐ |
| **`TI/`** | **라이브러리.** C2000Ware 버전 불일치로 인한 에러를 방지하기 위해 포함했습니다. | ⭐⭐⭐⭐ |
| **`targetConfigs/`** | **보드 연결 설정.** (`.ccxml`) 팀원들이 디버깅할 때 필요합니다. | ⭐⭐⭐ |
| **`.project`** | CCS 프로젝트 정의 파일 (숨김 파일). CCS에서 **Import** 할 때 필수입니다. | ⭐⭐⭐⭐⭐ |
| **`.cproject`** | CCS C/C++ 빌드 설정 파일 (숨김 파일). 컴파일 옵션 및 경로 정보가 포함됩니다. | ⭐⭐⭐⭐⭐ |
| **`.ccsproject`** | CCS 전용 설정 파일 (숨김 파일). | ⭐⭐⭐⭐⭐ |
| **`*.slx`, `*.prj`** | **Simulink 모델 및 프로젝트 파일.** (예: `SOFC_test1.slx`) MBD 환경 설정 파일입니다. | ⭐⭐⭐⭐⭐ |
| **`*.cmd`** | 링커 명령어 파일 (`280049_FLASH_lnk.cmd` 등). 메모리 맵 정의용입니다. | ⭐⭐⭐⭐ |
