## 📂 주요 폴더 및 파일 구성

이 프로젝트를 협업하거나 빌드하기 위해 필수적인 파일들의 목록과 중요도입니다.

| 폴더/파일 | 설명 | 중요도 |
| :--- | :--- | :--- |
| **`User/`** | **핵심 소스코드.** (`main.c`, `CTRL.c` 등) 프로젝트의 가장 중요한 부분입니다. | ⭐⭐⭐⭐⭐ |
| **`TI/`** | **라이브러리.** C2000Ware 버전 불일치로 인한 에러를 방지하기 위해 포함했습니다. | ⭐⭐⭐⭐ |
| **`targetConfigs/`** | **보드 연결 설정.** (`.ccxml`) 팀원들이 디버깅할 때 필요합니다. | ⭐⭐⭐ |
| **`.project`** | CCS 프로젝트 정의 파일 (숨김 파일). CCS에서 **Import** 할 때 필수입니다. | ⭐⭐⭐⭐⭐ |
| **`.cproject`** | CCS C/C++ 빌드 설정 파일 (숨김 파일). 컴파일 옵션 및 경로 정보가 포함됩니다. | ⭐⭐⭐⭐⭐ |
| **`.ccsproject`** | CCS 전용 설정 파일 (숨김 파일). | ⭐⭐⭐⭐⭐ |
| **`*.slx`, `*.prj`** | **Simulink 모델 및 프로젝트 파일.** (예: `SOFC_test1.slx`) MBD 환경 설정 파일입니다. | ⭐⭐⭐⭐⭐ |
| **`*.cmd`** | 링커 명령어 파일 (`280049_FLASH_lnk.cmd` 등). 메모리 맵 정의용입니다. | ⭐⭐⭐⭐ |


## 🚫 Git 제외 항목 (.gitignore)

다음 파일 및 폴더는 로컬 환경에 종속적이거나 빌드 시 자동 생성되므로, **절대 Git에 올리지 않도록 주의**해야 합니다.

| 폴더/파일 | 설명 | 제외 이유 |
| :--- | :--- | :--- |
| **`Debug/`** | 빌드 결과물 (`.obj`, `.out`, `.map`) | 빌드할 때마다 바뀌며 용량이 큽니다. 소스만 있으면 다시 만들 수 있습니다. |
| **`.settings/`** | 로컬 IDE 설정 | 사용자마다 화면 설정이나 폰트가 다를 수 있어 충돌(Conflict)이 자주 납니다. |
| **`.launches/`** | 디버그 실행 설정 | 내 컴퓨터의 절대 경로가 포함될 수 있어 팀원 컴퓨터에선 에러가 날 수 있습니다. |
| **`resources/`** | CCS 내부 캐시 | 자동 생성되므로 불필요합니다. |
| **`*.bak`** | 백업 파일 | 깃(Git) 자체가 백업 도구입니다. 저장소가 지저분해지니 뺍니다. |
| **`*.tmp`, `*.log`** | 임시 파일 | 불필요한 시스템 로그 및 임시 데이터입니다. |
